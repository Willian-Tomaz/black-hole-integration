# black-hole-integration

Tradução do código de integração numérica em Mathematica para Julia 

Como utilizar:
1 - Instale o Julia 
    https://julialang.org/downloads/

2- Instale os pacotes dentro do Julia 
    ArbFloats v0.3.2
    ArbNumerics v1.2.5
    DecFP v1.3.1
    Nemo v0.33.7
    Plots v1.38.10
    Readables v0.3.3

3- Faça 'julia two.jl"
